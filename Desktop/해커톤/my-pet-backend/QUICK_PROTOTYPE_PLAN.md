## 🎯 MVP 핵심 기능 (발표용)

### ✅ 필수 구현 (반드시)
1. **반려동물 프로필 관리** - 여러 마리 등록/선택
2. **증상 입력** - 텍스트 + 사진 업로드 (음성은 선택)
3. **멀티 에이전트 진료 시각화** - 4명의 AI가 순차적으로 진료하는 것처럼 보이기
4. **진단서 생성** - 응급도, 진단 결과, 조치사항
5. **직접 치료하기 / 병원 예약하기** - 버튼과 기본 UI
6. **병원 추천** - 카카오맵 연동 (실제 API 또는 모킹)
7. **마이페이지** - 진료 기록 목록

### ⚠️ 선택 구현 (시간 여유 시)
- 음성 입력 (녹음 기능)
- PDF 업로드
- 실제 AI API 연동 (초기에는 모킹 데이터)
- 병원용 기능

## 🛠 빠른 구현 전략

### 1. AI API는 나중에, 먼저 UI 완성
- **초기**: 하드코딩된 응답 데이터로 멀티 에이전트 시각화
- **나중**: 실제 OpenAI API 연동 (시간 여유 시)
- **장점**: UI/UX 완성도 높이고, API 연동은 독립적으로 진행 가능

### 2. 백엔드 없이 시작
- **로컬 스토리지**로 데이터 저장
- 프로필, 진료 기록 모두 로컬에 저장
- **장점**: 서버 구축 시간 절약, 빠른 프로토타입

### 3. 카카오맵은 실제 연동 (간단하게)
- 카카오 Local API로 병원 검색만
- 지도는 나중에 (시간 여유 시)
- **장점**: 핵심 기능은 유지하면서 시간 절약


### 3. 카카오맵 API (간단하게)
```javascript
// 카카오 Local API - 동물병원 검색
const searchHospitals = async (query, x, y) => {
  const response = await axios.get(
    `https://dapi.kakao.com/v2/local/search/keyword.json`,
    {
      params: { query: '동물병원', x, y, radius: 5000 },
      headers: { Authorization: `KakaoAK ${API_KEY}` }
    }
  );
  return response.data.documents;
};
```
